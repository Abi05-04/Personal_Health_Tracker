# 🏥 Health Tracker Web App (Streamlit)

A modern and interactive **Health Tracking Web Application** built using Streamlit.
Track your daily health metrics like **weight, calories, steps, and water intake** with real-time dashboards, charts, and export features.

---

## 🚀 Features

* 🔐 User Login System
* ➕ Add daily health data
* 📊 Real-time dashboard (metrics cards)
* 📈 Data visualization (line & bar charts)
* 📄 View all records in table format
* 🗑 Delete entries
* 📤 Export data to Excel
* 🌐 Clean and responsive web UI

---

## 🛠️ Tech Stack

* Python
* Streamlit
* SQLite
* Pandas
* OpenPyXL
* Matplotlib (for charts)

## ▶️ Run Locally

### 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR-USERNAME/health-tracker.git
cd health_streamlit
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run Application

```bash
streamlit run app.py
```
